## Getting Started

To get a local copy up and running follow this steps:

1. Clone the repository and `cd` into it

```sh
git clone https://github.com/DiegoBelPe/bank-occ-app.git
cd bank-occ-app
```

2. Install dependencies

```sh
npm install
```

3. Start the project

```sh
npm run dev
```

4. To build the site for production

```sh
npm run build
```
